常用國字標準字體表，簡稱甲表A4808字，次常用國字標準字體表，簡稱乙表B6329字，罕用國字標準字體表，簡稱丙表C18319字與「異體字字典」附錄編輯說明中描述字體相同，唯新增正字N「異體字字典」編輯說明中爲465字，而實際據「異體字字典」統計出來467字，故總字爲29923字，望悉知。
